# Práce s JSON ve VS Code

## Cíl

Seznámení se strukturou JSON souboru a hierarchickými daty.

---

## Co obsahuje soubor

Soubor `customers.json` obsahuje:

- objekt zákazníka,
- klíče a hodnoty,
- pole objednávek,
- boolean hodnoty,
- hierarchickou strukturu dat.

---

## Procvičené koncepty

- JSON syntaxe
- objekty {}
- pole []
- key-value struktura
- formátování JSON ve VS Code
- práce s datovými soubory

---

## Poznámky

JSON je vhodný pro:
- API,
- webové aplikace,
- přenos dat,
- hierarchická data.

Na rozdíl od CSV podporuje zanořené struktury.
